# freeTool v0.0.3

## Installation

Using npm:
```bash
$ {sudo -H} npm i -g npm
$ npm i --save freeTool
```

freeTool export 
```js
const fTool = {
  cookieUtil,
  Local,
  Session,
  menuTreeGenerator,
  ...method, 
}
```

See the [package source](https://github.com/FreemenL/freeTool) for more details.
