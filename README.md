# empty-tool v0.0.8
## Installation

Using npm:
```bash
$ npm i --save empty-tool
```
## Usage

### In webpack 
Need to use tsloader to handle empty-tool directory

empty-tool export 
```js
const fTool = {
  cookieUtil,
  Local,
  Session,
  menuTreeGenerator,
  ...method, 
}
```

See the [package source](https://github.com/FreemenL/empty-tool) for more details.
